# Habridio Kurniawan Putra — Portfolio

Personal portfolio site showcasing 7+ years of backend engineering experience at Blibli.com.

**Live:** https://habridio.dev

## Tech Stack

- [Next.js 16](https://nextjs.org) — App Router, SSR, sitemap & robots generation
- [Tailwind CSS v4](https://tailwindcss.com) — utility-first styling
- [Framer Motion](https://www.framer.com/motion/) — scroll-triggered animations
- [TypeScript](https://www.typescriptlang.org) — end-to-end type safety
- [Vercel](https://vercel.com) — deployment & hosting

## Local Development

```bash
npm install
npm run dev     # http://localhost:3000
npm run build   # production build
npm run lint    # ESLint
```
